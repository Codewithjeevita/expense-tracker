# 💰 Expense Tracker

A simple Expense Tracker built using Python.  
It helps users record and manage their daily expenses.

## 🚀 Features
- Add expense with date, category, and amount
- View all expenses
- Calculate total spending
- Simple menu-based program

## 🛠️ Technologies Used
- Python

## ▶️ Run
python expense_tracker.py
